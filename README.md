# bashcoder

AtCoder の問題の bash による解答例。
リポジトリにはコードを置いていません。
リンク先にコードがあるのでそれを見てください。

[Wiki][1] には bash で問題を解くために必要な知識などが書いてあります。

awk は原則として使わないと決めています。
そもそも awk だけで問題は解けるはずなので。
